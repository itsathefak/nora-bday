# NoraFlix

A private local birthday web app built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - reusable UI components
- `data/` - local JSON content for each profile
- `public/images/` - placeholder images
- `public/videos/` - placeholder video directories

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run start`

## Notes

The app includes:

- `/` profile selection screen
- `/profile/[profile]` placeholder profile pages
- `/watch/[type]/[id]` placeholder watch page

This is the first step of the NoraFlix app with a clean structure, responsive UI, and sample local data.
